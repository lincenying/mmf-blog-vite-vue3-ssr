First installation `api server`:

koa2: https://github.com/lincenying/mmf-blog-api-koa2-v2

express: https://github.com/lincenying/mmf-blog-api-v2

```bash

# Install dependencies
$ yarn
# or
$ npm install
# Note: do not install with CNPM dependency

# Product
$ yarn build

# Develop
$ yarn serve
```
